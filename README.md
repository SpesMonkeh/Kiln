# Tools
[Nodejs](http://nodejs.org/)

### Browser build :
```
yarn # npm install
yarn dev # npm run dev (npm run release for final build, npm run website should not be used)
// visit app/index.html
```

### Standalone build:
```
yarn add electron
yarn add electron-packager
yarn standalone
```
